SkillSync

Sync your skills with the perfect job.

or drag and drop

PDF, DOCX up to 10MB

This project is a full-stack application designed to provide intelligent feedback on resumes.

  • Backend: Built with Python and FastAPI, providing a high-performance API for all the analysis.
  • Natural Language Processing (NLP): We use the powerful spaCy library to understand the text from both the resume and job description.
  • Analysis Process:
    • First, we extract the raw text from your PDF or DOCX file.
    • Then, we calculate a semantic similarity score to determine the overall match percentage.
    • We perform a Skill Gap Analysis by comparing an extensive list of hard and soft skills found in the job description against your resume.
    • Finally, the system generates actionable tips based on common resume-writing best practices.
  • Frontend: A responsive and modern user interface built with HTML, Tailwind CSS, and vanilla JavaScript.
